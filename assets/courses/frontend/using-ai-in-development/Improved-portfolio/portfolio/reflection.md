## Ethical Challenges and Mitigation

During this project, I encountered several ethical challenges and learned how to address them effectively:

1. Difficulty understanding complex code implementations, which highlighted the importance of clear and concise explanations.

2. Instances where AI-generated code lacked proper documentation or failed to provide necessary import links, leading to implementation delays.

3. Situations where unintended outcomes occurred due to unclear prompts provided to the AI, emphasizing the need for precise communication.

To mitigate these issues, I focused on improving my prompt clarity, verifying AI-generated code thoroughly, and seeking additional resources to bridge knowledge gaps.
